# EnbuAP Battle Data

The Battle Data of The Game '幻想人形演舞-ユメノカケラ-' made by 'FocasLens'.

## Data Format

The data is stored by .csv format, endcoding with UTF-8 with BOM, which can be opened by Excel or many Text Editor (Notepad++, Sublime, Vim, Emacs ...).
